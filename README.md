# DanMu-springMVC

This is a springMVC test.

这个项目是关于SpringMVC的实践，实现的是简单的弹幕系统，可以通过扫描二维码和直接在[网页](http://zhonghuiblog.com:8080/DanMu/spring/danmu)的文本框中输入，来发射弹幕。

*** 

主要运用到的技术就是：Servlet3.1新增加的WebSocket，前端的页面布局和jQuery实现的弹幕效果
